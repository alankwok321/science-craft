# Science Craft ⚗️

An Infinite Craft-style game where you combine science elements to discover new ones. Start with Atom, Energy, Water, and Earth and work your way up to Black Holes, AI, and the Theory of Everything. 80+ elements to discover.

## How to Play

1. Drag elements from the sidebar onto the workspace
2. Combine two elements by dragging one onto another
3. Discover new elements and unlock the full tree of science!

## Play Online

Open `index.html` in any modern browser — no server required.
